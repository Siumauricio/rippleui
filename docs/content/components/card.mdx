# Card

Ripple UI comes with a card component that can be used to display content in a card format.

### Default

Simple example of a card.

<div className="card">
  <div className="card-body">Hello World</div>
 </div>

```html
<div className="card">
  <div className="card-body">Hello World</div>
 </div>
```

### Title, Body and Footer

A card can have a title, body and footer.

<div className="card">
  <div className="card-body">
      <h2 className="card-header">Hello World!</h2>
    <p className="text-content2">I really like the cats! lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
      <div className="card-footer">
    <button className="btn-secondary btn">Buy a Cat</button>
  </div>
  </div>
</div>

| ---------------- | -------------------------------------- |
| card             | Card base class                        |

| card-header      | Card header                            |
| card-body        | Card body                              |
| card-footer      | Card footer                            |
| card-border      | Will set border color                  |
