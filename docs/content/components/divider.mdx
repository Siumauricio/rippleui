# Divider

Ripple UI comes with component to separate your elements with a divider.

## Default

Simple example of Divider.

<div className="divider"></div>

```html
<div className="divider"></div>
```

## Horizontal Divider

Horizontal divider with text in the middle .

<div className="divider divider-horizontal">OR</div>

```html
<div className="divider divider-horizontal">OR</div>
```


## Vertical Divider

Vertical divider with text in the middle.

<div className="flex justify-center">
    <div className="divider divider-vertical h-44">OR</div>
</div>

```html
<div className="divider divider-vertical h-44">OR</div>
```

## Start Text Divider

Divider with text at the start.

<div className="flex items-center gap-1">OR<div className="divider divider-horizontal w-full"></div></div>

```html
<div className="flex items-center gap-1">
    OR
    <div className="divider divider-horizontal w-full"></div>
</div>
```

## End Text Divider

Divider with text at the end.

<div className="flex items-center gap-1"><div className="divider divider-horizontal w-full"></div>OR</div>

```html
<div className="flex items-center gap-1">
    <div className="divider divider-horizontal w-full"></div>
    OR
</div>
```

## With containers

Divider with containers.

<div className="flex w-full justify-center gap-4">
    <div className="card max-w-full items-center justify-center">Default</div>
    <div className="divider divider-vertical mx-0 h-44"></div>
    <div className="card max-w-full items-center justify-center">Default</div>
</div>

```html
<div className="flex w-full justify-center gap-4">
    <div className="card max-w-full items-center justify-center">Default</div>
    <div className="divider divider-vertical mx-0 h-44"></div>
    <div className="card max-w-full items-center justify-center">Default</div>
</div>
```

## API
<Attributes >
<Attributes.Title>Divider Classes</Attributes.Title>

| class                 | Description                                        |
| --------------------- | -------------------------------------------------- |
| divider               | Divider base class                                |
| divider-horizontal    | Set horizontal orientation                         |
| divider-vertical      | Set vertical orientation                           |

</Attributes>
