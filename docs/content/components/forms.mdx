## API
<Attributes>
<Attributes.Title>Form Classes</Attributes.Title>

| className                 | Description                   |
| --------------------- | ----------------------------- |
| form-group            | wrapper to agrupate a form                |
| form-field            | wrapper for a form-label, form-label-alt and form-control          |
| form-label            | add styles to a title of form element         |
| form-label-alt        | className to display any important information like error, success           |
| form-control          | wrapper for input, textarea, select, checkbox or any other form element           |
</Attributes>
