# Breadcrumb

Ripple UI comes with a breadcrumb component that helps you to show the current page in a hierarchy.

## Default

Simple example of a breadcrumb.

<div className="breadcrumbs text-sm">
  <ul>
    <li>
      <a>Home</a>
    </li>
    <li>
      <a>Library</a>
    </li>
    <li>Add Data</li>
  </ul>
</div>
