# Kbd

Ripple UI comes  with a keyboard component that allows you to display keyboard keys.

## Default

Simple example of a kbd.

<kbd className="kbd"> Space </kbd>


```html
<kbd className="kbd"> Space </kbd>
```

## Keys with text

You can combine text with keyboard keys.

Press <kbd className="kbd"> Space </kbd> + <kbd className="kbd"> Shift </kbd>  + <kbd className="kbd"> Alt </kbd> + <kbd className="kbd"> A </kbd> to open the application.

```html
Press <kbd className="kbd"> Space </kbd> + <kbd className="kbd"> Shift </kbd>  + <kbd className="kbd"> Alt </kbd> + <kbd className="kbd"> A </kbd> to open the application.
```

## Sizes

You can change the size of the keyboard keys.

    <kbd className="kbd kbd-md"> Space </kbd>
   <kbd className="kbd kbd-lg"> Space </kbd>
    <kbd className="kbd kbd-xl"> Space </kbd>
</div>
