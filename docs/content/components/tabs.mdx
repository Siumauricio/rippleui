# Tabs

Ripple UI comes with a set of tabs that can be used to organize content into different sections. They are used to switch between different views.

### Default

Simple example of a tabs.

<div className="tabs">
  <div className="tab">Tab 1</div>
  <div className="tab">Tab 2</div>
  <div className="tab tab-active ">Tab 3</div>
</div>

```html
<div className="tabs">
  <div className="tab p-4">Tab 1</div>
  <div className="tab p-4">Tab 2</div>
  <div className="tab tab-active p-4">Tab 3</div>
</div>
```

### Bordered

Bordered tabs.

<div className="tabs">
    <div className="tab tab-bordered px-6">Tab 1</div>
    <div className="tab tab-bordered tab-active px-6">Tab 3</div>
    <div className="tab tab-bordered px-6">Tab 3</div>
</div>

```html
<div className="tabs">
    <div className="tab tab-bordered px-6">Tab 1</div>
    <div className="tab tab-bordered px-6 tab-active">Tab 3</div>
    <div className="tab tab-bordered px-6">Tab 3</div>
</div>
```

### Underline

Underline tabs.

<div className="tabs">
    <div className="tab px-6">Tab 1</div>
    <div className="tab tab-underline tab-active px-6">Tab 2</div>
    <div className="tab px-6">Tab 3</div>
</div>

```html
<div className="tabs">
    <div className="tab">Tab 1</div>
    <div className="tab tab-underline tab-active">Tab 2</div>

</div>
```

### Pill

Pill tabs.

<div className="tabs gap-1">
    <div className="tab tab-pill">Tab 1</div>
    <div className="tab tab-pill tab-active">Tab 2</div>
    <div className="tab tab-pill tab-disabled">Tab 3</div>
</div>

```html
<div className="tabs gap-1">
    <div className="tab tab-pill">Tab 1</div>
    <div className="tab tab-pill tab-active">Tab 2</div>
    <div className="tab tab-pill tab-disabled">Tab 3</div>
</div>
```

<Attributes>
<Attributes.Title>Tabs Classes</Attributes.Title>

| class         | Description                               |
| ------------- | ----------------------------------------- |
| tabs          | Wrapper for tabs                          |
| tab           | Tab base style                            |
| tabs-boxed    | Will apply a background color to the tabs |
| tab-bordered  | All tabs will have a border by default    |
| tab-underline | Will apply border bottom to a element     |
| tab-pill      | Will have a hover and active effect       |
| tab-active    | Will mark a tab as active                 |
| tab-disabled  | Will disable the tab                      |



</Attributes>
