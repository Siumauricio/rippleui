# Default Theme

RippleUI comes with a default theme that you can use to build your website or application with no customization, when you toggle the toggle-icon you will see the light colors or dark colors, in the next sections you will see how to customize the theme.

<Attributes edit="/pages/en-us/components/button.mdx">

| Step | Case                                    |
| ---- | :-------------------------------------- |
| 50   | UI element background                   |
| 100  | Hovered UI element background           |
| 200  | Active / Selected UI element background |
| 300  | Subtle borders and separators           |
| 400  | Subtle borders and separators           |
| 500  | Hovered UI element border               |
| 600  | Solid backgrounds                       |
| 700  | Hovered solid backgrounds               |
| 800  | Low-contrast text                       |
| 900  | Low-contrast text                       |

</Attributes>
